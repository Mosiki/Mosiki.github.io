奔向杭州

2019-03-15
  • 其他

博客两年来，第一次断更，，，原因嘛，一是过年，二是确实上个月没有做技术方面的事情，导致自己想要写博客没有好的选题。毕竟也是坚持了两年多的写博客的习惯了，之后还是会补上，把这个习惯持续保持下去。

离职

在北京工作了两年的时间，逐渐觉得自己在技术上的提升变得特别慢，然后又有些泛于皮毛，技术的深度也不够。在公司里，也不需要加班，平时机动性地给其他项目写写业务代码，觉得自己慢慢地已经走入舒适区太久，也该跳出来看看了。

于是 3 月初开始投简历，奈何投完一圈竟都石沉大海，刷脉脉看到有 HR 说，发布一个只招 2人 的 Java 开发岗位，竟能接收到 2千 份以上的简历，才知，北京的 Java 市场竞争已是如此惨烈。

还好，有两家公司约我去面试，也是直接拿到了其中一家的 Offer，另外一家公司因为自己无心继续留在北京，在与 HR 谈薪资的时候，不肯让步，可能 HR 觉得我没有诚意吧，因而未给我发放 Offer。

由于其他的一些原因，不想再呆在北京，于是转向杭州投简历，杭州 Java 市场没有那样饱和，面试机会还是挺多的。

面试了几家，总结下来，觉得自己这几年真的是白混了 😂，无论公司大小，面试必问源码，包括不限于 JDK 源码、Dubbo 源码、Spring 源码，协议底层，但自己在这方面确实欠缺很多，可能面试我的都是阿里出来的吧，所以几乎都挂了，不过后面还是拿到了几家 Offer。薪资上和北京的相比，实话说差很多，但是权衡一下，毕竟还年轻，成长与发展更重要，于是决定推掉北京的 Offer，去杭州。

去杭州

决定了去杭州，一如既往地利索，找了德邦，把大部分的行李打包直接发往杭州，当晚就坐车去了杭州。

杭州对于我而言，又是一个新的开始，正如两年前去北京一般。

待惯了北京，习惯了北京便利的地铁，来到杭州后，去哪里都不是很方便，坐公交真是要命，还好杭州处处都在建地铁站，可想而知，过几年，交通方面也能方便很多。希望自己不会后悔这次抉择吧！

最后

当务之急，还是需要好好磨砺自身的技术，毕竟而立之年已不久矣！
